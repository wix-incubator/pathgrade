# Contributing

Thank you for your interest in Pathgrade.

This repository is published from Wix Incubator. At this time, external code contributions are not accepted unless Wix Security and legal teams define a different process for this repository.

Please do not open unsolicited pull requests with code changes. They may be closed without review.

For non-security feedback, use the public project channels that are enabled for this repository. For security issues or suspected vulnerabilities, do not open a public issue or pull request; follow the private reporting process in [SECURITY.md](SECURITY.md).
